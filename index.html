Twitter access

  • Look at conf/application.conf
  • http://apps.twitter.com/ - register
  • URL needs to start with http(s)://; also you need to verify with your phone num
  • Setup security tokens on heroku - Settings
  • Restart your app

Something more lively

  • Add var: TWITTER_FILTER_KEYWORD with value `java`
  • Restart your app

Now CD part - integrate Heroku with Travis

  • Heroku - your profile, API KEY
  • Travis - create variable named: HEROKU_API_KEY with this value
  • .travis.yml - deploy section
  • Push everything to master or PR
